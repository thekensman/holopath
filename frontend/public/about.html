About HoloPath

HoloPath was built out of frustration with the lack of good, free tools for creating hologram content. Most hologram pyramid videos on YouTube were made with expensive 3D software, and there was no simple way to take an existing image or video and turn it into something you could project through a DIY hologram pyramid.

HoloPath solves this by providing a free, browser-based generator that applies realistic holographic effects — scan lines, glow, chromatic aberration, glitch, noise, and more — to any image, GIF, or short video. The output is formatted specifically for hologram display hardware, not just for social media aesthetics.

What Makes HoloPath Different

Unlike simple Instagram-style hologram filters, HoloPath produces output in the correct layout for actual hologram hardware. It supports five display formats:

  • Pyramid 360° — the classic 4-sided layout for smartphone pyramid projectors
  • Pyramid 270° — 3-sided layout for commercial showcase displays
  • Hologram Fan — circular crop for LED persistence-of-vision fan displays
  • Pepper's Ghost — positioned for angled glass or acrylic reflectors
  • Single Image — direct hologram output for sharing or custom use

Each frame runs through a 13-stage rendering pipeline built with JavaScript and Canvas 2D, producing effects that look authentic rather than gimmicky. Five carefully tuned presets (Classic, Cyberpunk, Ghost, Glitch, Matrix) provide a starting point, and every parameter is adjustable.

How It Works

For static images (JPG, PNG, WebP, BMP), HoloPath generates animation by varying effects like scan line position, flicker phase, and glitch offsets across frames — creating the illusion of a live holographic display from a single photo.

For animated GIFs, each source frame is individually processed through the hologram pipeline, preserving the original animation while adding all the holographic effects.

For videos (MP4, WebM, MOV), frames are extracted in your browser via the HTML5 video element at 10fps, processed through the hologram pipeline, and output as an animated GIF. Longer videos use more browser memory but there is no hard duration limit.

Privacy

All processing happens entirely in your browser. Your files are never uploaded to a server or stored anywhere. HoloPath is a static website with no backend — there is no database, no user accounts, and no way for anyone to access your files. See the full Privacy Policy.
