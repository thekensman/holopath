Privacy Policy

Last updated: February 2026

Data Collection

HoloGen does not collect, store, or transmit any personal information. No account creation is required. No cookies are set by HoloGen itself (third-party services like Google AdSense may set their own cookies as described below).

File Handling

All processing happens entirely in your browser using client-side JavaScript. Files you select (images, GIFs, and videos) are read locally by your browser and are never uploaded to any server. The hologram rendering pipeline, layout compositing, and GIF encoding all run on your device. No file data ever leaves your computer.

HoloGen is a static website — it has no backend server, no database, and no file storage. The web server only delivers the HTML, CSS, and JavaScript files that make up the application.

Advertising

HoloGen uses Google AdSense to display advertisements. Google may use cookies and web beacons to serve ads based on your prior visits to this and other websites. You can opt out of personalised advertising by visiting Google's Ads Settings. For more information, see Google's privacy policy.

Analytics

HoloGen does not use Google Analytics or any other analytics tracking service. Server access logs (IP address, requested URL, timestamp) are retained for up to 14 days for security and debugging purposes, then automatically deleted via log rotation.

Third-Party Links

This site contains links to Buy Me a Coffee, Ko-fi, and Google. These external sites have their own privacy policies and HoloGen is not responsible for their content or practices.

Contact

For privacy-related questions, please reach out via the support links on the About page.
